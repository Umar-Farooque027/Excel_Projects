# 🏥 Hospital Emergency Room Dashboard (Excel)

## 📌 Project Overview
This project is an interactive Emergency Room Dashboard built using Microsoft Excel.  
It helps in analyzing patient flow, waiting time, and overall hospital performance in the emergency department.

## 🎯 Objective
- Monitor patient visits in the emergency room
- Analyze waiting time and service efficiency
- Identify peak hours and patient trends
- Support better decision-making for hospital management

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning & Transformation

## 📊 Key Metrics (KPIs)
- Total Patients
- Average Waiting Time
- Admission Rate
- Patient Distribution (Age / Gender)
- Peak Visit Hours

## 📈 Dashboard Features
- Interactive Filters (Date, Gender, Age Group)
- Dynamic Charts & Graphs
- Easy-to-understand Visualizations
- Real-time Insights from data
